# دفتر العملاء والخدمات

تطبيق Flutter عربي خفيف لإدارة العملاء والخدمات والدفعات بدون إنترنت.

## النسخة الأولى

- إضافة عميل مع رقم هاتف وملاحظة.
- إضافة خدمة للعميل مع تاريخ الخدمة أو الموعد.
- تسجيل مبلغ الخدمة والدفعة.
- حساب رصيد كل عميل تلقائيًا.
- ملخص عام: إجمالي الخدمات، المقبوض، والمبالغ المتبقية.
- صفحة تفاصيل لكل عميل تعرض خدماته وتسلسل العمليات.
- نسخ رسالة متابعة جاهزة للعميل.
- حفظ البيانات محليًا على الجهاز.
- تصميم عربي هادئ ومريح.

## طريقة البناء عبر GitHub

ارفع الملفات إلى GitHub، ثم شغل workflow باسم:

`Build APK Direct`

بعد نجاح البناء سيظهر ملف APK مباشرة داخل:

`apk/customers-services.apk`

وسيظهر ملف:

`DOWNLOAD_APK.md`

وفيه رابط التحميل المباشر.

## ملاحظات

- التطبيق Android فقط.
- لا يحتاج إنترنت أثناء الاستخدام.
- لا يحتاج صلاحيات حساسة.
- الحد الأدنى المقترح: Android 6 وما فوق.
